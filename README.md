# Информационная система продажи недвижимости

Курсовой проект на C++ с использованием Qt для управления недвижимостью.

## Описание

Система позволяет управлять каталогом недвижимости, клиентами и сделками. Поддерживаются три типа недвижимости:
- Квартиры
- Дома
- Коммерческая недвижимость

## Требования

- C++17 или выше
- CMake 3.16 или выше
- Qt6 (Core, Widgets)
- Компилятор с поддержкой C++17 (GCC 7+, Clang 5+, MSVC 2017+)

## Сборка

```bash
mkdir build
cd build
cmake ..
make
```

## Запуск

```bash
./EstateAgency
```

## Функциональность

### Управление недвижимостью
- Добавление, редактирование, удаление объектов недвижимости
- Поиск по адресу
- Фильтрация по цене
- Просмотр детальной информации

### Управление клиентами
- Регистрация клиентов с валидацией данных
- Поиск по имени или телефону
- Управление информацией о клиентах

### Управление сделками
- Создание сделок между клиентами и недвижимостью
- Отслеживание статуса сделок (в ожидании, завершена, отменена)
- История всех операций

### Работа с данными
- Автоматическое сохранение данных в файлы
- Загрузка данных при запуске
- Работа с файлами (потоки ввода/вывода)

## Валидация данных

Все входящие данные проверяются на корректность:
- Цены и площади должны быть положительными
- Телефоны проверяются по формату (+7XXXXXXXXXX или 8XXXXXXXXXX)
- Email проверяется регулярным выражением
- Паспорта проверяются по формату (XXXX XXXXXXX)
- Все обязательные поля должны быть заполнены
